# Coredump

启动用coredump:
```
ulimit -c unlimited
```

测试程序:
```
#include <iostream>

void fault_test(void) {
    *((int *)0x00000) = 100;
}

int main(int, char**) {
    std::cout << "Hello, world!\n";
    fault_test();
    std::cout << "endl" << std::endl;
    return EXIT_SUCCESS;
}
```

`CMakeLists.txt`:
```
cmake_minimum_required(VERSION 3.0.0)
project(ftrace_test VERSION 0.1.0)

add_executable(ftrace_test main.cpp)
```

编译并执行:
```
$ ./ftrace_test 
Hello, world!
段错误 (核心已转储)
```

查看`/var/lib/apport/coredump/`:
```
$ ls /var/lib/apport/coredump/

```
